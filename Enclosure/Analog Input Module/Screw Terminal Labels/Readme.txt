Note: Labels are only for Analog Input Module v2 or newer.

---Materials---
Printer: Brother PT-P900W
Label tape: TZe-M355 (Premium White Print on Black Matte)

---Print settings---
Print resolution: Standard
Output: Standard
Options: Half Cut, Output Directly to Printer Driver. ALL OTHERS UNCHECKED

---Instructions---
Print labels

5V and DIO: Cut to size on dotted lines.

AIN: DO NOT cut dotted lines dividing the channels. Instead, cut label length-wise across the tops and bottoms of the dotted lines.

Peel and stick on label space above wire entry holes on wire terminals.
