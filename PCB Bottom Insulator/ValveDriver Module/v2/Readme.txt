The design is optimized for printing on an Ultimaker 3 printer using black Colorfabb XT filament, and the following mods to the 'Normal' default slicing profile:

Layer height: 0.15mm
Wall thickness: 1.2mm
Top thickness: 1.2mm
Bottom thickness: 1.2mm
Infill: 40%
Infill pattern: Triangles
Printing temperature: 245 C
Build plate temperature: 75C (if glass + Kapton) or 50C (if PEI)
Print speed: 60mm/s
Fan speed: 50%
Use support: Yes
Support overhang angle: 70.0
Build Plate Adhesion: Skirt
